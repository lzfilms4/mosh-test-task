## Установка

1. Клонировать репозиторий:
    ```bash
    git clone <ссылка-на-репозиторий>
    ```
2. Перейти в папку проекта:
    ```bash
    cd mosh-test-task
    ```
3. Установить зависимости:
    ```bash
    npm install
    ```

## Запуск проекта

1. Запуск локального сервера:
    ```bash
    npm run dev
    ```

## Сборка проекта

Для создания production-сборки:

```bash
npm run build
```
